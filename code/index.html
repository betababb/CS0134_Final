About the guide:

The Astroneer Mini Manual was created as a final project for a website design and development class (CS0134) taught by Professor Olena Sherbinin at the University of Pittsburgh during Summer of 2022. This class has ignited my interest in website design, and I intend for this project to be the first of many. Any and all feedback is welcomed with open arms - you can find a form below this text to submit any comments you have directly to me, the creator. Thanks for visiting & happy exploring!
